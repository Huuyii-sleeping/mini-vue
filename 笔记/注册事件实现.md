# 事件注册的实现

就像click mousedown之类的事件

在h函数中进行实现

```ts
return h("div", {
            id: "root",
            class: ["red", "hard"],
            onClick(){
                console.log("click")
            },
            onMousedown(){
                console.log('mousedown')
            }
        },
```

我们在mount的时候进行响应的解析

这些都是props

```ts
const { props } = vnode
    for (const key in props) {
        const val = props[key]
        const isOn = (key) => /^on[A-Z]/.test(key)

        if (isOn(key)) {
            // 注册点击事件 具体 -> 通用
            // on+Event
            const event = key.slice(2).toLowerCase()
            el.addEventListener(event, val)
        } else {
            el.setAttribute(key, val)
        }

    }
```
